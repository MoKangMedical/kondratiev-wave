# 第45课：投资心理学与行为金融

> **学习目标**：认识投资中的认知偏差，掌握克服心理陷阱的实用方法。
> **预计时长**：40分钟
> **难度**：进阶
> **推荐书籍**：《思考，快与慢》《影响力》

## 一、开篇：为什么90%的散户亏钱？

**不是因为不够聪明，而是因为太"人性"了。**

卡尼曼和特沃斯基发现：人类的大脑天生不适合做投资决策。我们的直觉在远古时代帮助我们生存，但在金融市场中却不断制造错误。

| 认知偏差 | 表现 | 后果 |
|----------|------|------|
| 损失厌恶 | 亏损的痛苦是盈利快乐的2倍 | 止损太晚，止盈太早 |
| 锚定效应 | 被第一个价格锚定 | 高位追涨，低位杀跌 |
| 羊群效应 | 跟随大众 | 泡沫顶部买入 |
| 过度自信 | 高估自己的判断力 | 频繁交易 |
| 确认偏差 | 只看支持自己的信息 | 忽视风险信号 |

## 二、十大认知偏差详解

### 2.1 损失厌恶（Loss Aversion）

**定义**：失去100元的痛苦，是得到100元快乐的2倍。

| 场景 | 你的选择 | 理性选择 |
|------|----------|----------|
| 股票亏20% | 持有等回本 | 止损换更好的标的 |
| 股票赚20% | 赶紧卖掉锁定利润 | 持有让利润奔跑 |
| 打新股中签 | 立刻卖掉 | 看公司基本面决定 |

> **巴菲特名言**："如果你不愿意持有一只股票10年，那就不要持有10分钟。"

### 2.2 锚定效应（Anchoring）

- 房子挂牌价500万，你砍到450万觉得"赚了"——但实际只值400万
- 股票从100跌到50，你觉得"便宜了"——但它可能只值20
- **破解方法**：看估值（PE/PB），不看价格

### 2.3 羊群效应（Herd Behavior）

| 时期 | 羊群行为 | 结果 |
|------|----------|------|
| 2007年 | 全民炒股 | 2008年暴跌 |
| 2015年 | 杠杆牛 | 股灾 |
| 2021年 | 炒茅台/新能源 | 深度回调 |

> **核心洞察**：当出租车司机都在谈论股票时，就是该卖出的时候。

### 2.4 过度自信（Overconfidence）

- 散户平均年换手率500%+（机构约100%）
- 70%的散户认为自己能跑赢市场
- 实际上：80%的散户跑输指数

### 2.5 确认偏差（Confirmation Bias）

- 买了某股票后，只看利好新闻
- 忽视负面信息，强化正面信息
- **破解方法**：主动寻找反面论据

## 三、市场情绪指标

### 3.1 VIX恐慌指数

| VIX值 | 市场情绪 | 投资含义 |
|-------|----------|----------|
| <15 | 极度贪婪 | 谨慎 |
| 15-25 | 正常 | 中性 |
| 25-35 | 恐慌 | 开始关注 |
| >35 | 极度恐慌 | 可能是底部 |

> **巴菲特名言**："别人贪婪时我恐惧，别人恐惧时我贪婪。"

### 3.2 恐贪指数（Fear & Greed Index）

- 0-100的综合指标
- 综合波动率、看涨看跌比、市场宽度等
- 极端值往往是反转信号

## 四、投资清单系统

### 4.1 买入前检查清单

| 检查项 | 通过标准 |
|--------|----------|
| 估值合理 | PE/PB在历史中位数以下 |
| 基本面健康 | ROE>15%，负债率<60% |
| 行业趋势 | 处于上升周期 |
| 仓位控制 | 单只不超过总资产15% |
| 心态检查 | 不是FOMO，是理性分析 |

### 4.2 卖出前检查清单

| 检查项 | 卖出信号 |
|--------|----------|
| 基本面恶化 | ROE连续下降 |
| 估值过高 | PE超过历史80%分位 |
| 更好机会 | 有性价比更高的标的 |
| 止损触发 | 亏损超过预设阈值 |

## 五、克服心理陷阱的7个方法

1. **写投资日记**：记录每次买卖的理由和情绪
2. **设定规则**：提前设好止损止盈，严格执行
3. **分散投资**：不把鸡蛋放一个篮子
4. **定期复盘**：每月回顾投资决策
5. **逆向思考**：主动寻找反面论据
6. **延迟决策**：冲动时等待24小时
7. **模拟盘练习**：先用虚拟资金测试策略

## 六、课后练习

### 交互练习题

1. **识别偏差**：小王买了一只股票，从10元涨到15元后立刻卖出，结果股票继续涨到25元。他后悔不已，又在22元买回，结果跌到18元。请分析小王犯了哪些认知偏差？

2. **VIX分析**：当前VIX指数为40，市场恐慌。有人说"应该卖出所有股票"，有人说"应该大胆买入"。你认为哪个更合理？为什么？

3. **清单设计**：请为你自己设计一个"买入股票前的检查清单"，至少包含5个检查项和具体标准。

4. **情绪管理**：你持有的股票连续3天大跌，总共亏损15%。你的第一反应是什么？你会采取什么行动？请描述你的决策过程。

5. **案例分析**：2021年"茅指数"（茅台等白马股）从高点下跌50%以上。很多人在高点买入是因为"核心资产永远涨"。请分析这种思维的偏差。

1. **识别偏差**：①损失厌恶（赚了就怕跌，赶紧卖）；②锚定效应（被15元的价格锚定，22元买回时觉得"便宜了"）；③后悔偏差（后悔卖出导致冲动买回）；④追涨杀跌（涨了追买，跌了恐慌）。

2. **VIX分析**：VIX=40表示极度恐慌。历史数据表明，VIX>35时往往是市场底部区域。"大胆买入"更合理，但需要分批建仓而非一次性抄底。"卖出所有"是最差决策——在恐慌中卖出往往卖在最低点。

3. **清单设计**：示例——①估值：PE<行业中位数；②基本面：ROE>15%，连续3年增长；③行业：处于上升周期；④技术面：不在历史最高位；⑤仓位：单只<总资产10%；⑥心态：不是因为别人推荐，是自己分析的结果。

4. **情绪管理**：第一反应可能是恐慌和想卖出。正确做法：①暂停操作，不看盘24小时；②回顾买入理由是否改变；③检查止损点是否触发；④如果没有改变，坚持持有；③如果基本面恶化，果断止损。

5. **案例分析**：偏差包括：①近因效应（过去几年涨得好，以为会永远涨）；②从众心理（所有人都在买）；③过度自信（认为自己能选中"核心资产"）；④忽视均值回归（估值过高必然回归）。
