## 二、50门课程核心知识图谱

| 模块 | 课程 | 核心一句话 |

| 周期理论 | 1-6 | 经济有四季，顺势而为 |

### 2.2 十大核心原则

1. **周期不可违**：顺势而为，不逆周期
2. **复利是王道**：时间是最大的盟友
3. **分散是保险**：不把鸡蛋放一个篮子
4. **估值是锚**：好公司也要好价格
5. **情绪是敌人**：理性决策，纪律执行
6. **风险是朋友**：理解风险，管理风险
7. **学习是投资**：最好的投资是投资自己
8. **现金流为王**：被动收入决定自由度
9. **长期主义**：短期波动不影响长期趋势
10. **知行合一**：知道做不到等于不知道

## 三、投资体系的四大支柱

### 3.1 认知层——你相信什么？

| 信念 | 你的回答 |
|------|----------|
| 市场是否有效？ | 部分有效 |
| 能否跑赢市场？ | 可以，但需要纪律 |
| 风险和收益的关系？ | 正相关，但可以优化 |
| 预测市场是否可能？ | 短期不可，长期有规律 |

### 3.2 策略层——你怎么做？

**投资策略模板**：

| 资产类别 | 目标比例 | 允许波动范围 | 具体标的 |
|----------|----------|-------------|----------|
| 权益类 | 50% | 40-60% | A股ETF+港股+美股 |
| 实物资产 | 20% | 15-25% | 黄金+REITs |
| 固收类 | 20% | 15-25% | 国债+高等级债 |
| 现金 | 10% | 5-15% | 货币基金 |

### 3.3 执行层——你如何操作？

**投资纪律清单**：

| 规则 | 具体标准 |
|------|----------|
| 定投 | 每月15日定投，金额=月收入×30% |
| 再平衡 | 每季度检查，偏离>5%时调整 |
| 止损 | 单只亏损>20%时止损 |
| 止盈 | 单只盈利>50%时减半 |
| 加仓 | VIX>35时加大权益配置 |

### 3.4 风控层——你如何保护自己？

**风控清单**：

| 风险类型 | 应对措施 |
|----------|----------|
| 系统性风险 | 保持20%现金+黄金 |
| 流动性风险 | 不投资锁定期>1年的产品 |
| 集中度风险 | 单只不超过总资产15% |
| 杠杆风险 | 永不使用杠杆 |
| 对手方风险 | 选择大型合规机构 |

## 八、课后练习（终极练习）

### 交互练习题

1. **宣言撰写**：请根据50门课程所学，撰写你的个人投资宣言。包括：投资目标、投资原则、资产配置、投资纪律。

2. **策略设计**：假设你有100万元，30岁，请设计一个完整的投资方案。包括：资产配置比例、具体标的、再平衡规则、风控措施。

3. **复盘练习**：回顾你过去3年的投资经历，用行为金融学的知识分析你犯过哪些认知偏差。

4. **清单制作**：制作你自己的"投资决策检查清单"，至少包含10个检查项。

5. **终极思考**：如果只能给未来的自己写一封信，关于投资，你会说什么？
