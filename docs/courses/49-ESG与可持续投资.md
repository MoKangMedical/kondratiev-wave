# 第49课：ESG与可持续投资

> **学习目标**：理解ESG投资的逻辑，掌握绿色金融和碳交易的投资机会。
> **预计时长**：40分钟
> **难度**：进阶
> **推荐书籍**：《ESG投资》《气候经济与人类未来》

---

## 一、开篇：投资不仅要赚钱，还要对得起良心？

**ESG = Environment（环境）+ Social（社会）+ Governance（治理）**

这不是情怀，而是新的投资范式。全球ESG资产规模已超过35万亿美元。

| ESG维度 | 关注点 | 投资影响 |
|---------|--------|----------|
| E - 环境 | 碳排放、污染、资源使用 | 高碳企业被抛弃 |
| S - 社会 | 员工权益、社区影响 | 丑闻导致股价暴跌 |
| G - 治理 | 管理层诚信、股东权利 | 治理差的公司风险高 |

## 四、ESG基金表现

### 4.1 ESG基金vs传统基金

| 指标 | ESG基金 | 传统基金 |
|------|---------|----------|
| 长期收益 | 相当或略高 | 基准 |
| 波动率 | 略低 | 基准 |
| 最大回撤 | 较小 | 较大 |
| 尾部风险 | 较低 | 较高 |

> **核心洞察**：ESG不是牺牲收益做慈善，而是通过规避风险获得更好的风险调整后收益。

## 五、漂绿（Greenwashing）

### 5.1 什么是漂绿？

把不符合ESG标准的产品包装成"绿色"、"可持续"。

### 5.2 识别漂绿的方法

| 红旗 | 说明 |
|------|------|
| 数据不透明 | 无法验证ESG数据 |
| 只讲故事 | 没有具体指标 |
| 名称误导 | "绿色基金"实际重仓石化 |
| 绿色收入占比低 | <10%的绿色收入 |

## 七、课后练习

### 交互练习题

1. **评估题**：一家公司声称自己是"绿色企业"，但其收入中只有5%来自清洁能源，95%来自煤炭。这是ESG投资还是漂绿？

2. **分析题**：中国碳市场碳价从50元涨到100元/吨。这对哪些行业有利？对哪些行业不利？

3. **投资题**：你有10万元想投资ESG主题，会在光伏ETF、新能源车ETF、碳中和ETF之间如何分配？

4. **判断题**：ESG基金的收益一定比传统基金低吗？为什么？

5. **思考题**：如果一家公司ESG评分很高但估值很贵（PE=80），你还愿意投资吗？

## 参考答案

<details><summary>点击查看参考答案</summary>

1. **评估题**：这是典型的漂绿。绿色收入占比仅5%，95%仍是高碳业务。真正的ESG企业应该有>30%的绿色收入，且有明确的转型计划。

2. **分析题**：有利——新能源、碳捕集、林业碳汇、碳交易平台。不利——钢铁、水泥、化工、煤电等高碳行业。碳价上升直接增加高碳企业的成本。

3. **投资题**：建议——光伏ETF 40%（确定性最高）+新能源车ETF 35%（增长最快）+碳中和ETF 25%（主题覆盖广）。理由：光伏产业链中国优势明显，新能源车渗透率仍有空间。

4. **判断题**：不一定。ESG基金通过规避环境风险、社会丑闻、治理问题，可以获得相当甚至更好的风险调整后收益。研究表明，ESG评分高的公司长期股价表现优于同行。

5. **思考题**：需要综合考虑。ESG好+估值贵=好公司但好价格？建议：①如果成长性确定（如光伏龙头），高PE可以接受；②如果成长性不确定，等待估值回调；③可以用定投方式分散估值风险。
