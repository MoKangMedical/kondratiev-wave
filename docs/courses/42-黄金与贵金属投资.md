# 第42课：黄金与贵金属投资

> **学习目标**：理解黄金的货币属性与投资逻辑，掌握贵金属投资的核心策略。
> **预计时长**：40分钟
> **难度**：进阶
> **推荐书籍**：《货币战争》《大贬值》

## 一、开篇：为什么黄金3000年不败？

**黄金是人类历史上唯一持续保值的资产。**

| 时期 | 1盎司黄金能买 | 2026年等价 |
|------|---------------|------------|
| 古罗马 | 一件优质长袍 | ~$2,000 |
| 1900年 | 一套定制西装 | ~$2,000 |
| 2026年 | 一套定制西装 | ~$2,300 |

> **核心洞察**：黄金不产生收益，但它永远不归零。在法币贬值的大背景下，黄金是最后的货币。

## 二、黄金的货币属性3000年

### 2.1 为什么是黄金？

| 特性 | 说明 |
|------|------|
| 稀缺性 | 地壳含量0.004ppm |
| 不可伪造 | 密度19.3g/cm³，难以仿制 |
| 可分割 | 可以熔铸成任意大小 |
| 不腐蚀 | 永不氧化变质 |
| 便携性 | 价值密度高 |

### 2.2 金本位制的兴衰

- **1816年**：英国正式采用金本位
- **1944年**：布雷顿森林体系，美元与黄金挂钩（$35/盎司）
- **1971年**：尼克松关闭黄金窗口
- **1971-1980年**：黄金从$35涨到$850（24倍）
- **1980-2000年**：黄金熊市，从$850跌到$250
- **2001-2011年**：黄金牛市，从$250涨到$1,900
- **2024-2026年**：突破$2,300，央行大量购入

## 三、黄金价格的驱动因素

### 3.1 四大驱动力

| 驱动因素 | 影响方向 | 当前状态 |
|----------|----------|----------|

|------|----------------|------|

## 四、贵金属投资方式

### 4.1 投资方式对比

| 方式 | 优势 | 劣势 | 适合人群 |
|------|------|------|----------|
| 实物金条 | 真正持有 | 存储成本 | 长期持有者 |
| 黄金ETF | 流动性好 | 纸黄金风险 | 短中期交易 |
| 黄金期货 | 杠杆放大 | 爆仓风险 | 专业交易者 |
| 金矿股 | 带杠杆+分红 | 经营风险 | 成长型投资者 |
| 白银 | 工业属性+弹性 | 波动更大 | 进取型投资者 |

### 4.2 黄金vs白银

| 指标 | 黄金 | 白银 |
|------|------|------|
| 主要属性 | 货币属性 | 工业+货币 |
| 金银比 | 1 | 1/80 |
| 波动率 | 低 | 高 |
| 新能源需求 | 低 | 高（光伏） |

## 六、中国大妈抢金的经济学分析

**2013年"中国大妈"抢金事件**：

- 金价暴跌至$1,400/盎司
- 10天内购入300吨黄金
- **短期看**：抄底抄在半山腰（金价继续跌至$1,050）
- **长期看**：2024年金价突破$2,300，大妈们最终解套

> **教训**：黄金适合定投，不适合一次性抄底。长期持有者永远是对的。

## 七、课后练习

### 交互练习题

1. **计算题**：如果2026年金价$2,300/盎司，你用10万元人民币购买实物黄金（汇率7.2），能买多少盎司？如果2030年金价涨到$3,500，你的收益率是多少？

2. **分析题**：当前金银比为80:1，历史均值为60:1。如果金银比回归均值，白银价格应该是多少？这意味着白银有什么投资机会？

3. **决策题**：你有50万元可用于贵金属投资，你会如何在黄金、白银、金矿股之间分配？请给出具体比例和理由。

4. **判断题**：有人说"比特币将取代黄金成为数字时代的避险资产"。你同意吗？从货币属性、波动性、监管三个维度分析。

5. **案例题**：2022年俄乌冲突期间，黄金短暂上涨后迅速回落。为什么地缘风险没有持续推高金价？这对你的投资策略有什么启示？

## 参考答案

1. **计算题**：10万元÷7.2÷2,300≈6.04盎司。如果涨到$3,500，价值=6.04×3,500×7.2=152,208元，收益52,208元，收益率52.2%。

2. **分析题**：如果金银比回归60:1，白银价格=2,300÷60≈$38.3/盎司。当前白银约$28.75，上涨空间约33%。白银的工业需求（光伏）提供了额外支撑。

3. **决策题**：建议分配——黄金60%（30万，核心保值）+白银25%（12.5万，弹性更大）+金矿股15%（7.5万，杠杆效应）。理由：黄金提供稳定性，白银提供弹性，金矿股提供杠杆。

4. **判断题**：不同意。比特币波动性太大（年化80%+），无法充当避险资产；监管不确定性高；但比特币在"数字黄金"叙事上有一定合理性，适合小比例配置（5%以内）。

5. **案例题**：黄金短暂上涨后回落是因为：①美联储加息预期压制金价；②冲突未扩大到全球级别；③流动性紧缩导致抛售。启示：地缘事件是短期催化剂，不是长期驱动力，不要追涨。
