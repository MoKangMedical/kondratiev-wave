# 附录B：宋鸿兵《货币战争》系列核心框架

> 宋鸿兵，2007-2017年，中信出版社，系列累计销量超千万册
> 全系列五卷：《货币战争货币战争①-⑤》

## B.1 宋鸿兵其人

宋鸿兵，国际金融研究者，曾在美国金融机构工作。2007年出版《货币战争》引爆中国财经图书市场，系列累计销量超千万册，是将西方金融史通俗化的第一人。

**核心命题**：货币不是中性的工具，而是**权力的武器**。现代金融体系的本质是**私人银行通过控制货币发行权来控制国家**。

## B.2 五卷核心论点

### 卷①：金钱权族（2007）
**核心论点**：现代金融体系的核心不是政府，而是**私人银行家族**

- 以罗斯柴尔德家族为线索，揭示"金融王朝"如何通过控制货币发行权掌控全球经济
- 美联储（Federal Reserve）并非联邦机构，而是**私人银行联合体**
- 八大金融家族：罗斯柴尔德、洛克菲勒、摩根、杜邦等如何通过联姻和控股形成金融寡头网络
- "我不管谁当总统，谁控制货币谁才是真正的主人"——罗斯柴尔德

**关键概念**：
- **金本位 vs 法币**：金本位限制了银行的货币创造能力，所以金融精英推动去金本位
- **部分准备金制**：银行只需保留10%存款即可放贷100%，本质上是"无中生有"创造货币
- **私有央行**：美联储1913年成立，私人股东持有，总统无权干预

### 卷②：金权天下（2009）
**核心论点**：美元霸权是美国全球统治的根基

- 布雷顿森林体系（1944）：美元与黄金挂钩，其他货币与美元挂钩 → 美元=世界货币
- 1971年尼克松关闭黄金窗口 → 美元脱钩黄金，成为纯信用货币
- 石油美元体系：OPEC以美元计价石油 → 全球必须持有美元 → 美元需求永恒
- 美国通过"印钞"收割全球：用纸换商品，通胀由全世界承担

**关键概念**：
- **特里芬难题**：美元作为世界货币，必须持续逆差来提供全球流动性，但逆差又削弱美元信用
- **铸币税**：发行货币的国家从持有该货币的人那里"隐形收税"
- **美元周期**：美联储通过加息/降息制造全球资本流动，收割新兴市场

### 卷③：金融高边疆（2011）
**核心论点**：金融是"第四战场"，货币战争是国家间最高形态的战争

- 历史上所有大国兴衰都与货币体系有关
- 英国→美国的权力转移本质是英镑→美元的货币霸权转移
- 日本1985年广场协议：被迫让日元升值 → 泡沫经济 → 失去三十年
- 1997亚洲金融危机：索罗斯攻击泰铢 → 东南亚经济倒退十年

**关键概念**：
- **货币攻击**：通过做空一国货币引发资本外逃、经济崩溃
- **金融制裁**：现代战争不必动武，切断SWIFT即可瘫痪一国经济
- **金融主权**：一个国家必须拥有独立的货币发行权和金融基础设施

### 卷④：战国时代（2013）
**核心论点**：全球货币体系正在从"美元独霸"走向"战国群雄"

- 人民币国际化：跨境贸易结算、离岸市场、加入SDR
- 区域货币合作：亚投行、金砖银行
- 欧元的困境：统一货币但不统一财政 → 欧债危机
- 数字货币的可能：比特币等去中心化货币挑战传统金融

### 卷⑤：山雨欲来（2017）
**核心论点**：全球债务泡沫终将破裂，新的货币体系即将到来

- 全球债务总量超过GDP的300%，远超2008年水平
- 负利率是"金融异象"，标志着传统货币政策失效
- 黄金将回归货币体系核心——各国央行增持黄金是信号
- "最终的清算"不可避免：要么债务重组，要么货币重置

## B.4 货币战争视角下的当前局势（2026年）

### 美元体系的裂缝

宋鸿兵在卷⑤（2017年）的预测正在兑现：

1. **去美元化加速**
   - 金砖国家推动本币结算
   - 各国央行黄金增持量创历史新高（2023-2025）
   - 人民币跨境支付系统（CIPS）交易量激增

2. **美国债务危机信号**
   - 联邦债务超35万亿美元
   - 债务利息支出超国防预算
   - 美元信用边际递减

3. **黄金回归**
   - 黄金价格突破历史高位
   - "黄金和美元同时走强"（时寒冰洞察）
   - 与宋鸿兵"黄金将回归货币核心"的预测吻合

### 货币战争视角下的投资启示

| 判断 | 逻辑 | 投资含义 |
|------|------|---------|

| 黄金长期走强 | 货币体系重置的"锚" | 核心配置15-20% |
| 人民币国际化 | CIPS+金砖合作+数字货币 | 关注人民币资产 |

## B.5 宋鸿兵的"货币重置"预测

宋鸿兵在卷⑤中提出：当前全球货币体系（美元信用本位）最终将发生**重置**，可能的路径：

### 路径一：渐进式（概率较高）
- 美元缓慢衰落，多极货币体系逐步形成
- 黄金作为"货币之锚"角色增强
- 区域货币联盟扩大（亚元？金砖货币？）
- 时间跨度：20-30年

### 路径二：危机式（概率中等）
- 美国债务危机突然爆发
- 美元暴跌，全球金融市场剧烈动荡
- 紧急召开新版"布雷顿森林会议"
- 时间跨度：可能在任何一次康波萧条的"余震"中触发

### 路径三：技术式（概率渐增）
- 数字货币（央行数字货币CBDC或加密货币）重塑货币体系
- 去中心化货币挑战国家货币垄断
- 区块链技术使跨境支付绕过SWIFT
- 时间跨度：与第六轮康波同步

### 对投资的含义

无论哪条路径，**黄金都是"无条件受益者"**：
- 渐进式 → 黄金缓慢上涨
- 危机式 → 黄金暴涨
- 技术式 → 黄金作为"最后的锚"仍有价值

这也是时寒冰将黄金列为七大方向之首、康波理论将黄金列为萧条期最佳资产的**三重共振**。
