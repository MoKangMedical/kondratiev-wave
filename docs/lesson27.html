# 第27课：激荡三十年 — 中国经济的奇迹与教训

💡 核心洞察

推荐书籍：《激荡三十年》《激荡十年》（吴晓波）

📖 开篇：中国经济的奇迹

1978年，中国人均GDP只有156美元。2023年，超过12000美元。

这是人类历史上最大规模的经济奇迹。40年间，8亿人脱贫，中国成为世界第二大经济体。

但奇迹的背后，也有无数的失败、泡沫和教训。

📖 四个关键阶段

2.1 1978-1992：破冰

• 农村改革（家庭联产承包责任制）

• 乡镇企业崛起

• 个体户出现

• 价格双轨制

教训：改革是渐进的，不是一蹴而就的。

2.2 1992-2002：市场化

• 邓小平南巡

• 股票市场建立

• 国企改革（抓大放小）

• 亚洲金融危机

教训：市场化带来效率，但也带来风险。

2.3 2002-2012：全球化

• 加入WTO

• 房地产崛起

• 制造业成为世界工厂

• 2008年四万亿刺激

教训：全球化带来机遇，但也带来依赖。

2.4 2012-2022：新常态

• 经济增速换挡

• 互联网经济爆发

• 供给侧改革

• 中美贸易摩擦

教训：高速增长不可持续，高质量发展才是出路。

📖 历史给我们的投资启示

4.1 顺势而为

• 1990年代：买制造业股票

• 2000年代：买房地产

• 2010年代：买互联网股票

• 2020年代：买新能源/AI股票

4.2 警惕泡沫

• 2007年A股6000点

• 2015年创业板泡沫

• 2021年核心资产泡沫

4.3 控制杠杆

• 恒大的教训：过度杠杆=死亡

• 个人的教训：不要借钱炒股

📖 核心概念总结

你需要记住的5个关键点

1. 中国经济奇迹是渐进式改革的结果

2. 每10年都有一次大的产业变迁

3. 顺势而为是投资的核心

4. 泡沫和危机是周期性的

5. 控制杠杆是生存的基础
