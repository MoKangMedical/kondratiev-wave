# 第23课：债务危机 — 达里奥的全天候思维

💡 核心洞察

推荐书籍：《债务危机》（达里奥）、《原则》（达里奥）

📖 开篇：为什么债务危机反复发生？

2008年金融危机、2020年疫情冲击、2022年地产暴雷——债务危机每10-15年就会来一次。

达里奥研究了过去100年的48次债务危机，发现它们都有相同的模式：

1. 借钱容易→债务增长快于收入

2. 泡沫形成→资产价格虚高

3. 泡沫破裂→债务违约

4. 去杠杆→经济收缩

关键洞察：债务危机不是意外，而是必然。问题是：你准备好了吗？

📖 达里奥的债务周期模型

2.1 短期债务周期（5-8年）

• 央行降息→借贷增加→经济扩张→通胀上升→央行加息→借贷减少→经济收缩

2.2 长期债务周期（50-75年）

• 债务累积→泡沫→危机→去杠杆→复苏→新一轮债务累积

关键洞察：短期周期叠加在长期周期上。当两者同时向下时，就是大萧条。

2.3 当前位置

• 美国：长期债务周期的晚期（债务/GDP超130%）

• 中国：长期债务周期的中期（债务/GDP约300%）

• 全球：低利率时代结束，高债务+高利率=压力

📖 美丽的去杠杆

3.1 什么是"美丽的去杠杆"？

达里奥提出，去杠杆有四种方式：

方式效果例子
紧缩减少支出希腊
债务违约坏账出清雷曼兄弟
财富转移向富人征税皮凯蒂建议
印钞稀释债务美联储QE

|------|------|------|

美丽的去杠杆 = 四种方式的平衡组合。不能太紧缩（导致萧条），也不能太宽松（导致通胀）。

3.2 美丽去杠杆的条件

• 央行有足够工具（利率>0，有QE空间）

• 债务以本币计价（可以印钱还）

• 政府有财政空间（可以发债）

• 社会稳定（不会动乱）

📖 普通人如何应对债务危机

4.1 危机前的准备

• 减少负债（尤其是高息负债）

• 增加现金储备

• 分散资产（不把鸡蛋放一个篮子）

• 保持流动性（能随时变现的资产）

4.2 危机中的机会

• 优质资产被错杀时是买入机会

• 现金为王，等待抄底

• 不要恐慌性卖出

4.3 危机后的布局

• 经济复苏初期买股票

• 通胀初期买实物资产

• 新技术萌芽期买成长股

📖 核心概念总结

你需要记住的5个关键点

1. 债务危机是周期性的，每10-15年必然发生

2. 短期周期叠加长期周期，当两者同时向下时就是大危机

3. 美丽的去杠杆需要平衡紧缩、违约、转移和印钞

4. 危机前要减少负债、增加现金

5. 危机中是买入优质资产的最佳时机

你需要打破的3个误区

误区1："债务危机可以避免"

→ 不可以。它是由人性驱动的——贪婪和恐惧的循环。

误区2："印钞可以解决一切"

→ 印钞可以缓解短期痛苦，但会导致长期通胀和贫富分化。

误区3："危机中应该恐慌卖出"

→ 恐慌卖出是最差的策略。历史证明，危机后3-5年市场都会创新高。
