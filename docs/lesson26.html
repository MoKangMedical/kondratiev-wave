# 第26课：薛兆丰经济学讲义 — 用经济学思维看世界

💡 核心洞察

推荐书籍：《薛兆丰经济学讲义》

📖 开篇：经济学不是教你赚钱，而是教你思考

为什么钻石比水贵，但水比钻石重要？

薛兆丰说：经济学的核心就是一句话——价格是由供需决定的，不是由成本决定的。

理解这句话，你就理解了90%的经济学。

📖 价格理论：经济学的核心

2.1 价格是如何形成的？

因素影响例子
需求增加价格上涨疫情期间口罩涨价
供给减少价格上涨猪瘟导致猪肉涨价
需求减少价格下跌新能源车普及导致油价承压
供给增加价格下跌芯片产能扩张导致芯片降价

|------|------|------|

关键洞察：价格是稀缺性的信号。价格高=稀缺，价格低=丰富。

2.2 为什么"免费"的东西最贵？

• 免费医疗→排队3个月

• 免费教育→学区房天价

• 免费停车→找不到车位

原理：当价格被压制时，竞争会以其他形式出现（排队、关系、运气）。

📖 机会成本：经济学的第一思维

3.1 什么是机会成本？

做任何选择的成本，不是你付出了什么，而是你放弃了什么。

例子：

• 读研究生的成本不是学费，而是2年的工资

• 买房的成本不是月供，而是这笔钱投资股市的收益

• 排队买奶茶的成本不是奶茶钱，而是你30分钟的时间价值

3.2 如何用机会成本思维做决策？

1. 列出所有选项

2. 计算每个选项的机会成本

3. 选择机会成本最低的选项

📖 边际思维：经济学的第二思维

4.1 什么是边际思维？

不是看"总量"，而是看"下一个"。

例子：

• 吃第一个包子很满足，第十个很痛苦

• 第一桶金很重要，第一百万只是数字

• 第一次加班有动力，第十次想辞职

4.2 边际成本与边际收益

• 继续做下去的额外收益 > 额外成本 → 继续

• 继续做下去的额外收益 < 额外成本 → 停止

应用：

• 该不该加班？→ 加班的边际收益（加班费、晋升机会）vs 边际成本（健康、时间）

• 该不该投资？→ 投资的边际收益（预期回报）vs 边际成本（风险、流动性）

📖 核心概念总结

你需要记住的5个关键点

1. 价格由供需决定，不是由成本决定

2. "免费"的东西最贵，因为竞争会以其他形式出现

3. 机会成本是做决策时最重要的思维方式

4. 边际思维告诉你什么时候该停止

5. 经济学不是教你赚钱，而是教你思考
