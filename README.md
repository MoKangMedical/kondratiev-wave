# 康波周期投资预测体系 v1.0

> 基于康德拉季耶夫长波理论 + 周金涛"涛动周期论" + 中美顶级家族跨周期策略

## 📁 项目结构

```
kondratiev-wave/
├── README.md                          # 本文件
├── docs/
│   ├── 01_k_wave_theory.md            # 康波理论完整框架
│   ├── 02_zhou_jintao_framework.md    # 周金涛四周期嵌套体系
│   ├── 03_sixth_wave_analysis.md      # 第六轮康波（AI驱动）深度分析
│   ├── 04_family_strategies.md        # 中美顶级家族跨周期策略
│   └── 05_investment_playbook.md      # 投资操作手册（2026-2040）
├── data/
│   ├── k_wave_history.json            # 五轮康波历史数据
│   ├── asset_rotation.json            # 各阶段资产轮动规律
│   └── family_portfolios.json         # 顶级家族投资组合解构
└── web/
    ├── index.html                     # 交互式查询系统
    └── app.js                         # 前端逻辑
```

## 🎯 核心功能

1. **周期定位**：当前处于哪一轮康波的哪个阶段
2. **资产轮动查询**：不同阶段应该配置什么资产
3. **家族策略学习**：顶级家族如何跨越周期
4. **个人投资时间表**：根据年龄和周期定位生成投资建议
5. **大趋势雷达**：哪些行业/技术是第六轮康波的核心驱动力

## 📖 理论来源

- 康德拉季耶夫（Nikolai Kondratiev）：长波理论原始提出者（1925）
- 约瑟夫·熊彼特（Joseph Schumpeter）：创造性毁灭 + 三周期嵌套
- 周金涛：四周期嵌套 + "人生就是一场康波"（2016演讲）
- 《涛动周期论》《人生财富靠康波》
