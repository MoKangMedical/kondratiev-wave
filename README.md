## 🎯 产品定位

康波周期投资预测平台是一款**基于长波周期理论的宏观经济预测工具**，将四位经济预测大师的理论体系融合为可操作的投资决策框架：

| 理论维度 | 代表人物 | 核心问题 | 理论来源 |
|---------|---------|---------|---------|

| **趋势方向** | 时寒冰 | 买什么？ | 《未来二十年经济大趋势》《全球视野下的投资机会》 |

## 🧠 理论框架

### 康波周期：50-60年的经济长波

```
技术发明 → 基建投资 → 产能扩张 → 债务积累 → 泡沫破裂 → 萧条出清 → 新技术发明
```

| 轮次 | 时间跨度 | 主导技术群 | 持续年限 |
|------|---------|-----------|---------|
| 第一轮 | 1780-1845 | 蒸汽机 + 纺织机械 + 铁路 | ~65年 |
| 第二轮 | 1845-1900 | 钢铁 + 铁路网 + 电报 | ~55年 |
| 第三轮 | 1900-1950 | 电气化 + 化工 + 汽车 | ~50年 |
| 第四轮 | 1950-1990 | 石油化工 + 喷气机 + 电子 | ~40年 |
| 第五轮 | 1990-2030 | 信息技术 + 互联网 + 移动通信 | ~40年 |
| **第六轮** | **2025-2070** | **AI + 生物科技 + 新能源** | **~45年** |

### 当前定位（2026年4月）

```
第五轮：████████████████████░░ 萧条末期（95%完成）
第六轮：██░░░░░░░░░░░░░░░░░░░ 回升初期（5%完成）
```

我们正处于新旧康波交替的**"黎明前"阶段**——第五轮清算基本完成，第六轮技术种子已经播下。2026-2030是**播种期**。

### 四维共振框架

```
        周金涛（何时）
           ▲
          / \
         /   \
        /     \
时寒冰（何地）——宋鸿兵（为何）
```

- **周金涛**：我们在康波的哪个阶段？→ 决定大方向

- **宋鸿兵**：货币环境和金融体系如何演变？→ 决定估值框架
